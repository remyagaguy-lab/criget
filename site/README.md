# CRIGET-CS Site Web Vitrine

Projet Astro configuré pour Cloudflare Pages, tel que défini dans le cahier des charges.

## Démarrage rapide

1. Installez les dépendances :
   \`\`\`bash
   npm install
   \`\`\`

2. Lancez le serveur de développement :
   \`\`\`bash
   npm run dev
   \`\`\`

## Structure
- \`src/pages/fr/\` : Version française (Phase 1)
- \`src/pages/en/\` : Version anglaise (Phase 2 - structure réservée)
